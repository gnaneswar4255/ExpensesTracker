
# Expense Tracker

## How to Run the Project

### Backend
1. Navigate to `backend/`
2. Run `npm run dev`

### Frontend
1. Navigate to `frontend/`
2. Run `npm run dev`
3. And then press 'o'

## Technologies Used
- React, Tailwind CSS
- Node.js, Express.js
- MongoDB Atlas with Mongoose
- JWT for authentication

Refer to the documentation for more details.
